# Stack 계산기

## 후위 표기법 변환 1

1. 수식의 각 연산자에 대해서 우선순위에 따라 괄호를 사용하여 다시 표현
2. 각 연산자를 그에 대응하는 오른쪽 괄호의 뒤로 이동
3. 괄호 제거

---

## 후위 표기법 변환 2 (Stack 사용)

1. 입력 받은 중위 표기법에서 토큰 읽음
2. 토큰이 피연산자면 토큰 출력
3. 토큰이 연산자 일 때
- 이 토큰이 스택의 top에 저장되어 있는 연산자보다 우선순위가 높으면 스택에 Push
- 그렇지 않다면 스택 top의 연산자의 우선순위가 토큰의 우선순위보다 작을 때까지 스택에 pop 한 후 토큰의 연산자를 push한다.
- 만약에 top에 연산자가 없으면 push
1. 토큰이 오른쪽 괄호 (’ ) ’)이면 스택 top에 왼쪽 괄호 (’ ( ‘)가 올 때까지 스택에 pop 연산 수행 후 출력 왼쪽 괄호를 만나면 pop만 하고 출력 X
2. 중위 표기법에 더 읽을 것이 없다면 중지, 더 있다면 1번부터 다시 반복
3. 스택에 남아있는 연산자를 모두 pop하여 출력
- 스택 밖의 왼쪽 괄호는 우선순위가 가장 높음
- 스택 안의 왼쪽 괄호는 우선순위가 가장 낮음

ex) ( 6 + 5 * (2 - 8) / 2  →  6528-*2/+

## 후위 표기법 연산 (Stack 이용)

1. 피연산자를 만나면 스택에 push
2. 연산자를 만나면 필요한 만큼 피연산자를 스택에서 pop하여 연산하고 결과를 다시 스택에 push
3. 수식이 끝나면 마지막으로 스택을 pop하여 출력

ex)  6528-*2/+

- 6 5 -6 ( 2 - 8) → 6 -30 ( 5 * -6) → 6 -30 2 → 6 -15 ( -30 / 2) → -9

# 문자열 (String)

- 모든 문자들이 순서대로 나열된 데이터
- 텍스트(문장, 단어, 기호 등)을 표현하거나 처리할 때 사

---

## Length-Controlled 문자열

- 문자열의 길이 정보를 함께 저장해서, 그 길이만큼 문자 데이터를 읽는 방식
- Java, Python, 네트워크 패킷에 사용

---

## Delimited 문자열

- 문자열의 끝을 나타내는 특정한 구분자가 있어서, 구분자가 나올 때까지 문자열로 인식
- C 언어는 널문자(null, ‘\0’)을 사용

---

## 파이썬 str 클래스 구조

- 길이 외에 다른 정보도 저장
- PyObject_HEAD : 모든 Pyhton 객체가 상속하는 공통 구조
- length : 문자열의 길이
- hash : 문자열의 해시값으로, 딕셔너리 키를 쓸 때 사용
- interned : 같은 문자열을 관리하는 플래그
- kind : 문자열 인코딩의 크기
- data : 문자열이 저장된 실제 메모리 주소를 가리키는 포인

---

## C언어에서의 문자열

- 문자열은 문자들의 배열 형태로 구현된 응용 자료형
- 문자배열에서 문자열을 저장할 때는 항상 마지막 끝을 표시하는 널문자(’\0) 필요
- 문자열 처리에 필요한 연산을 함수 형태로 제공
    - strlen(), strcpy(), strcmp()

---

## Python3에서의 문자열

- 텍스트 데이터의 취급 방법이 통일되어 있음
    - python2와 달리 바이트 문자열과 Unicode 구분이 없음
    - 유니코드 기반이라 어떤 언어나 기호도 동일한 방식으로 처리
- 문자열 기호
    - ‘, ‘’, ‘’’,”””
- 연산
    - + 연결 :
        - 문자열 + 문자열을 이어 붙여주는 역
        - ‘ab’ + ‘c’
    - * 반복
        - 문자열 * 수 : * 수만큼 문자열이 반복
        - ‘ab’ * 3

## Python3에서의 문자열

- 문자열은 데이터의 순서가 구분되는 시퀀스 자료형으로 분ㄹ됨
    - 시퀀스 자료형에서 사용할 수 있는 인덱싱, 슬라이싱 연산들을 사용할 수 있음
- 문자열 클래스에서 제공되는 메소드
    - replac(), split(), isalpha(), find(0
- 문자열은 튜플과 같이 요소값을 변경할 수 없음

```python
# 인덱스 사용 가능
s = 'abc'
print(s[1])
```

```python
# 요소값 변경 불가능
s = 'abc'
s[1] = 'a' # TypeError: 'str' object does not support item assignment
```

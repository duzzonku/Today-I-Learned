# 표현식과 값

## 표현식 (Expression)

- 하나의 ‘값’으로 평가될 수 있는 모든 코드
- 표현식을 평가하면 하나의 값
- ex) 3+5 ,  x > 10, 5 * 4

---

## 값 (Value)

- 표현식이 평가된 결과
- 더 이상 계산되거나 평가될 수 없는, 프로그램의 가장 기본적인 데이터 조각
- 모든 값은 그 자체로 가장 단순한 형태의 표현식
- 모든 표현식은 값 X
- ex) 숫자 값 (103.14), 문자열 값 (”안녕하세요”), 불리언 값(True, False)

---

# 변수와 메모리

## 변수 (Variable)

- 값을 나중에 다시 사용하기 위해, 그 값에 붙여주는 고유한 이름

---

## 변수 할당 (Variable assignment)

- 표현식에 만들어 낸 값에 이름을 붙이는 과정 (연결)

---

## 할당문 ( Assignment Statement)

- degrees = 36.5
- degrees : 변수 이름
- = : 할당 연산자 → 오른쪽의 값을 왼쪽 변수에 ‘할당’한다.
- 36.5 : 표현식

---

## 변수명 규칙

- 영문 알파벳, 언더스코어(_), 숫자로 구성
- 숫자로 시작 X
- 대소문자를 구분
- 파이썬의 내부 예약어로 지정된 변수는 사용 X

---

## 변수, 값 그리고 메모리

- 메모리의 모든 위치에는 그 위치를 고유하게 식별하는 메모리 주소 존
- 메모리 주소 : 고유한 ID
- 타입 (Type) : 제품의 종류
- 값 (Value) : 제품의 실제 내용물
- 값 + 타입 + 주소 정보 = 객체 (Object)
- 변수 : 특정 객체를 ‘가리키는’ 이름표
- 변수는 메모리 주소를 가지지 않음

---

## 변수 (Variable)

- 값을 나중에 다시 사용하기 위해, 그 값에 붙여주는 고유한 이름
- 객체를 가리키는 이름

---

## 할당문 동작 순서

### Variable = expression

1. 오른쪽 표현식 평가
- 할당 연산자의 오른쪽에 있는 표현식 전체를 계산
- 하나의 결과값 (객체)를 만듦

1. 왼쪽 변수명 확인
- 이름이 처음 사용되었다면 → 새로운 ‘이름표’ 준비
- 이미 존재하는 이름 → 기존 ‘이름표’ 그래도 사용

1. 변수명과 결과값 연결
- 왼쪽 변수명이 오른쪽에서 만들어진 결과값 가리키도록 연결
- 변수명이 이전에 다른 객체를 가리켰다면, 새로운 객체와의 연결만 남음

---

## 재할당 (Reassignment)

- 이미 값이 할당된 변수에 새로운 값을 다시 할당하는 것
- 변수가 가리키는 대상을 새로운 값으로 변경하는 행위
- 이전 값을 완전히 잊고, 새로운 값만 기억

## 데이터 타입 (Data Types)

### Sequence Types

- list, tuple, range

### Non-sequence Types

- set, dict

### Text Sequence Type

### 기타

# Sequence Types

- 여러 개의 값들을 순서대로 나열하여 저장하는 자료형
- 대표 시퀀스 타입 : str, list, tuple, range

---

## 시퀀스 타입의 5가지 공통 특징

1. **순서**
- 값들이 순서대로 저장 (정렬 X)
1. **인덱싱**
- 각 값에 고유 번호(인덱스)를 가지고 있다
- 특정 위치의 값을 선택하거나 수정 가능
1. **슬라이싱**
- 인덱스 범위를 조절해 원하는 부분만 값을 잘라내서 사용 가능
1. **길이**
- len() 함수 사용
- 저장된 값의 개수를 구할 수 있음
1. **반복**
- 반복문을 사용하여 각 값을 하나씩 순서대로 꺼내 사용 가능

## 문자열 만들기

### 1. 따옴표 활용

- 작은따옴표(’) 또는 큰따옴표(’’)로 감싸서 표현

```python
# 문자열 표현
print('Hello, World!')  # Hello, World!
print(type('Hello, World!'))  # str
```

- 따옴표 안에 다른 따옴표를 넣고 싶을 때는 서로 다른 종류의 따옴표 사용

```python
# 중첩 따옴표
print(
    '문자열 안에 "큰따옴표"를 사용하려면 작은 따옴표로 묶는다.'
)  # 문자열 안에 "큰따옴표"를 사용하려면 작은 따옴표로 묶는다.
print(
    "문자열 안에 '작은따옴표'를 사용하려면 큰따옴표로 묶는다."
)  # 문자열 안에 '작은따옴표'를 사용하려면 큰따옴표로 묶는다.
```

### 3. 특수 표현 - 이스케이프 시퀀스

- 역슬래시와 문자를 조합해 특별한 기능 수행
- ex) 줄바꿈, 탭

```python
# 따옴표 앞에 \를 붙여 문자로 인식시킴
print('He\'s a boy.')
# \n 은 줄바꿈(엔터)을 의미함
print('첫째 줄\n둘째 줄')

# 여러 줄 문자열을 작성할 때는 """ 또는 '''를 사용
multi_line_str = """
이것은
여러 줄로 이루어진
문자열입니다.
"""
print(multi_line_str)
```

- \n : 줄 바꿈
- \t : 탭
- \\ : 백슬래시
- \’ : 작은 따옴표
- \’’ : 큰 따옴

# 문자열에 값 삽입하기 : f-string

## f-string

- 문자열 내에 변수나 표현식의 결과를 손쉽게 삽입하는 방법
- 문자열 시작 전 ‘f’ 접두어를 붙이고 삽입할 부분을 중괄호 {}로 감싸줌

```python
# String Interpolation "f-string"
name = '홍길동'
age = 25
greeting = f'안녕하세요, 제 이름은 {name}이고 나이는 {age}살입니다.'
# 안녕하세요, 제 이름은 홍길동이고 나이는 25살입니다.
print(greeting)
```

# 시퀀스로서의 문자열

- 문자열은 시퀀스이므로 인덱싱, 슬라이싱, 길이 확인, 반복 등 공통 기능 모두 사용 가능

---

## 인덱스

- 시퀀스 자료형에서 각 값의 위치를 식별하기 위해 부여된 고유한 번호
- 0부터 시작
- 음수 인덱스 지원 - 끝에서부터 값에 접근

---

## 슬라이싱

- 시퀀스의 일부분을 잘라내어 새로운 시퀀스를 만드는 작업
- 시작, 끝 인덱스를 지정해 해당 범위의 값을 포함하는 새로운 시퀀스 생성

## 슬라이싱 사용 방법

### my_sequence[ start : stop : step]

- 대괄호 [ ] 안에 시작 위치, 끝 위치, 간격을 콜론{ : }으로 구분하여 지정
- start : 슬라이싱을 시작할 인덱스 (포함됨)
- stop : 슬라이싱을 끝낼 인덱스 (포함 X)
- step : 몇 개를 건너뛰며 값을 가져올지에 대한 간격

```python
# 문자열의 시퀀스 특징
my_str = 'hello'
# 1. 인덱싱
print(my_str[1])  # e

# 2. 슬라이싱
print(my_str[2:4])  # ll
print(my_str[:3])  # hel
print(my_str[3:])  # lo
print(my_str[::2])  # hlo
print(my_str[::-1])  # olleh

# 3. 길이
print(len(my_str))  # 5
```
